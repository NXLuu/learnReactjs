# Fragments
> Mẫu phổ biến cho React để trả về nhiều elements. Fragment giúp nhóm thành 1 list các children mà ko thêm node thừa vào DOM

```jsx
render() {
  return (
    <React.Fragment>
      <ChildA />
      <ChildB />
      <ChildC />
    </React.Fragment>
  );
}
```

## Motivation
## Short syntax
```jsx
class Columns extends React.Component {
  render() {
    return (
      <>
        <td>Hello</td>
        <td>World</td>
      </>
    );
  }
}
```

## Keyed Fragments
Fragments khai báo với <React.Fragment> sẽ có keys. 
```jsx
function Glossary(props) {
  return (
    <dl>
      {props.items.map(item => (
        // Without the `key`, React will fire a key warning
        <React.Fragment key={item.id}>
          <dt>{item.term}</dt>
          <dd>{item.description}</dd>
        </React.Fragment>
      ))}
    </dl>
  );
}
```
